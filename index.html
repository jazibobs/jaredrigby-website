Welcome!

I'm Jared Rigby, a Cheshire based web and applications developer. I love to write code which solve problems and firmly believe no task is too great given enough time and tea.

I was most recently working for Cognoscenti Solutions in Altrincham as a front and back-end web developer building a database driven health and safety document management system.

My humble site is powered by a Raspberry Pi which sits downstairs next to my router. It's a pretty neat piece of kit which I'm having a lot of fun with, plus I get to pretend to be a very important Linux system admin...

It is the aim of this site to provide a stage for me to show you guys what I code in my spare time. So please take some time to check out my current projects and the blog.
